# Silence TSUMUGI

## 概要

大島紬をテーマにしたジェンダーレスファッションブランドのECサイトです。

デザインからコーディング、フロントエンド実装まで一貫して制作しました。
Vue.jsを使用した商品一覧・ショッピングカート機能を実装し、GitHub Pagesで公開しています。

## ポートフォリオ

🔗 https://naosato-four.github.io/portfolio/

## 使用技術

- HTML5
- CSS3
- JavaScript (ES6)
- Vue.js 2
- Bootstrap 5
- GitHub
- GitHub Pages
- localStorage

## 主な機能

- 商品一覧表示
- カラーバリエーション切り替え
- ショッピングカート
- 数量変更
- 商品削除
- 注文ページ
- 注文完了ページ
- レスポンシブ対応

## 工夫した点

- 大島紬の世界観を表現するため、余白・タイポグラフィ・配色にこだわり、ブランドサイトとしてのデザインを意識しました。
- Vue.jsを利用して商品情報・カート情報を管理し、リアクティブな画面更新を実装しました。
- localStorageを利用し、ページを再読み込みしてもカート情報が保持されるようにしました。
- GitHub Pagesへデプロイし、実際に閲覧・操作できる状態で公開しています。
- 開発中に発生した404エラーやVue.jsのテンプレートエラーについては、ブラウザの開発者ツールを用いて原因を切り分け、一つずつ解決しました。
